How Testing Access Works

ParentShield testing builds are currently distributed through private or manual access rather than through a public website download.

This helps protect the build while development is still active and keeps distribution more controlled during testing.

If accepted into testing, your build link or APK file will be provided directly by Ninja Riot Studio.

Contact About Testing

If you have already submitted a testing request and need help, use the contact page and include the email you used.

What Happens Next

  1. Submit your testing request through the site.
  2. If testing is a fit, Ninja Riot Studio will contact you with next steps.
  3. Your APK file or private build link may be provided manually.
  4. Download the APK to your Android device.
  5. Open the APK from your device and follow Android's install prompts.

Testing Notes

ParentShield is an active testing build under development. Features, design, and functionality may change over time.

This version may be distributed directly by Ninja Riot Studio before broader release.

Testing access does not grant ownership, partnership, or decision-making authority in Ninja Riot Studio or ParentShield.

Need Help?

If you have trouble receiving a testing build, installing the APK, or opening ParentShield, contact Ninja Riot Studio and we will help you get set up.
